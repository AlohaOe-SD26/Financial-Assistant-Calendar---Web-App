## ğŸ“Š Understanding Merged Items

When you merge items, the amounts **ADD TOGETHER**:

**Example:**
```
Person A: Rent $1,500
Person B: Rent $1,500
Merged:   Rent $3,000 (Combined)
```

This is correct because:
- Total household rent is $3,000
- Shared by both people
- Shows accurate total expense

**For separate items:**
```
Person A: Car Payment $300 (Person A)
Person B: Car Payment $400 (Person B)
Not merged: Each shown separately
```

This is correct because:
- Each person has their own car
- Not a shared expense
- Individual responsibility
